# Processamento Digital de Imagens (PDI)

Este repositório contém as atividades desenvolvidas em laboratório na disciplina de **Processamento Digital de Imagens (PDI)**, do curso de **Ciência da Computação** do **IFNMG**.

## Objetivo

O objetivo deste repositório é organizar e documentar as práticas realizadas em aula, demonstrando a aplicação dos conceitos teóricos de PDI por meio de implementações em Python.

## 🛠️ Tecnologias utilizadas

* Python
* NumPy
* Matplotlib
